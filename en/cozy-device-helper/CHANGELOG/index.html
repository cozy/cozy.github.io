    4.0.0 (2025-09-05)¶

    Features¶

    • cozy-device-helper: Remove cordova related methods (1b963c4)

    BREAKING CHANGES¶

    • cozy-device-helper: Every cordova related methods have been removed.

    When updating, if you find a method below, it is probably dead code that you can remove. :

    • hasDevicePlugin,
    • hasInAppBrowserPlugin,
    • hasSafariPlugin,
    • hasNetworkInformationPlugin
    • checkApp
    • startApp
    • nativeLinkOpen
    • openDeeplinkOrRedirect
    • getDeviceName

    When updating, if you find a method below, keep what is relevant, i.e. what is related to isWebApp() or !isMobileApp() or !isCordova() :

    • isCordova
    • getPlatform
    • isIOSApp
    • isAndroidApp
    • isMobileApp
    • isWebApp
